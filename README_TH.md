### üìê ‡∏™‡∏π‡∏ï‡∏£‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì EQI (Embedding Quality Index)

```
EQI = (0.4 √ó S + 0.35 √ó H_score + 0.25 √ó info_ratio) √ó 100
```

‡πÇ‡∏î‡∏¢:
- **S** = Signal Quality Score (0-1)
- **H_score** = Entropy Score ‡∏ó‡∏µ‡πà‡∏õ‡∏£‡∏±‡∏ö‡πÅ‡∏•‡πâ‡∏ß (optimal ~ 0.7)
- **info_ratio** = effective_dims / total_dims

**‡∏Å‡∏≤‡∏£‡∏ï‡∏µ‡∏Ñ‡∏ß‡∏≤‡∏° EQI:**
- 70-100: ‡∏î‡∏µ‡∏°‡∏≤‡∏Å (Good)
- 50-70: ‡∏õ‡∏Å‡∏ï‡∏¥ (Normal)
- 30-50: ‡∏Ñ‡∏ß‡∏£‡∏ï‡∏£‡∏ß‡∏à‡∏™‡∏≠‡∏ö (Warning)
- 0-30: ‡∏°‡∏µ‡∏õ‡∏±‡∏ç‡∏´‡∏≤ (Bad)
